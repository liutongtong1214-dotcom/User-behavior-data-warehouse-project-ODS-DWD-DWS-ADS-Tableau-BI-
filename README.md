# User Behavior Data Warehouse Project

## Project Overview
This project builds a user behavior data warehouse based on e-commerce logs,
including ODS, DWD, DWS and ADS layers, and visualizes key metrics using Tableau.

## Tech Stack
- MySQL
- Python (pandas, SQLAlchemy)
- Tableau
- GitHub

## Data Warehouse Layers
- ODS: Raw user behavior data
- DWD: Cleaned and standardized behavior data
- DWS: Subject-oriented aggregated tables
- ADS: Application-level metrics (DAU, MAU, Conversion, Retention)

## Key Metrics
- DAU / MAU
- Conversion Rate
- User Retention (1d / 7d)
- Top 10 Hot Items

## Visualization
Built BI dashboards using Tableau for user behavior analysis.
